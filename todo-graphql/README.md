## How to run

* For client run the following commands:
    ```
       cd client
       yarn install
       yarn start
    ```

Application will be available on http://localhost:3000

* For server run the following commands:
    ```
       cd server
       npm install
       npm start
    ```
Application will be running on http://localhost:3003
